About the Project

Urban Sound Classification

The Urban Sound Classifier is a deep learning application designed to identify and classify various urban environmental sounds. This project uses advanced audio processing techniques and neural network architectures to accurately recognize sounds like car horns, sirens, street music, drilling, and other common urban noises.

Model Architecture

Our classifier uses a Double U-Net architecture with attention mechanisms to process audio spectrograms. The model is trained on the UrbanSound8K dataset, which contains over 8,000 labeled sound excerpts from urban environments.

Features

  • Multi-class sound classification
  • Real-time audio processing
  • Mel spectrogram visualization
  • Support for multiple audio formats
  • User-friendly web interface
